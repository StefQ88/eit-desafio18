## Configuración
1. Copia `.env.example` a `.env`
2. Completa las variables:
   - `<USER>`: usuario de Atlas
   - `<PASSWORD>`: contraseña del usuario (no la subas al repo)
   - `<HOST>`: p.ej. `admin.eii7li6.mongodb.net`
   - `<DB>`: p.ej. `tasksdb`
3. Instala y levanta:
   ```bash
   npm i
   npm run dev
